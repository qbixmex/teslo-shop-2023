# Teslo Shop

Website developed with NextJs

## Run development

```bash
#npm
npm run dev

# yarn
yarn dev
```

## Build application

```bash
# npm
npm run build

# yarn
yarn build
```

## Start application

**IMPORTANT: You must run build before run this script**

```bash
# npm
npm run start

# yarn
yarn start
```

## Run Lint

```bash
# npm
npm run lint

# yarn
yarn lint
```
